# claude-automation

## 概要
Claude Code デスクトップ版用の自動起動・自動応答スクリプト集。
tmux + auto-responder で Claude Code を自動操作していた旧環境。

## 現在の状況
VSCode版 Claude Code に移行したため、このスクリプトは基本不要。
過去の実装記録として保管。

## スクリプト
- `claude-start.sh` - 起動スクリプト（tmux + Claude Code + 自動応答）
- `claude-stop.sh` - 停止スクリプト
- `auto-responder.sh` - 自動応答ロジック
- `start-claude-session.sh` - セッション起動
